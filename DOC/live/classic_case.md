# 

POLYV保利威直播适用于各类音视频直播场景，比如教育行业的在线教育、网络课堂、双师课堂等；医疗行业的医学在线培训、手术直播、医学会议等； 娱乐行业的带货直播、游戏直播等；金融行业的企业在线培训、投资者教育、大会直播、一对一投资顾问等；另外还适用于展会直播、微信直播、年会直播、企业培训等其他视频相关行业。
